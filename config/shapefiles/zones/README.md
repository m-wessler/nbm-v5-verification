# Shapefiles Directory

This directory should contain shapefiles for NWS Public Forecast Zones.

## Expected Files

Place zone shapefiles here. Typical files include:
- `*.shp` - Shapefile geometry
- `*.shx` - Shape index
- `*.dbf` - Attribute table
- `*.prj` - Projection information

## Data Sources

Zone shapefiles can be obtained from:
- National Weather Service GIS data portal
- NOAA's public data repositories

## File Naming

Files should follow a consistent naming convention for the verification system to recognize them.
